[Thymeleaf]
- core th:xx is replaced by thymeleaf
- <html xmlns:th="http://www.thymeleaf.org">
- th:href="@{/css/style.css}"
- url link literal - @{}
    - thymeleaf use @{} for URL link.
    - th:href="@{/css/style.css}"
- literal replace |...|
    - "hello " + user.name
        - without literal, <span th:text="hello " + ${user.name}>
        - with literal, <span th:text="|hello ${user.name}|">
    - "location.href='/basic/items/add'"
        - without literal, "'location.href=' + '\'' + @{'/basic/items/add'} + '\''"
        - with literal, "|location.href='@{/basic/items/add}'|"
- th:each
    - <tr th:each="item : ${items}">
- variable expression - ${...}
    - <td th:text=""${item.price}"></td>
- th:text
    - change text by variable
    - <td th:text="${item.price}">
- th:action
    - form action same url("/")

    - redirect url
